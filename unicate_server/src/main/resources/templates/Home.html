Welcome! This is a web application where students can input a list of school names to output the calendars for those schools. The program will also compare the available vacation times on the
calendars to output a list of dates during which school vacation days overlap. The program will also check the surrounding locations for each school to find potential meeting locations.
Users are required to login to save the output information and make posts about their vacation locations.
